# Brainstorming Notes

Drop your brainstorming README files and notes here.

This folder is for early-stage thinking, design explorations, meeting notes, and any other informal documents that informed the POC design but aren't part of the formal design documentation.

Suggested naming convention:
- `YYYY-MM-DD-topic.md` for date-stamped notes
- `topic-exploration.md` for design explorations
- `meeting-YYYY-MM-DD.md` for meeting notes
